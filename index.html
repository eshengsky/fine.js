fine.js

A fine, tiny and practical JavaScript utility library, less than 5KB with compression.

一个精致、小巧、实用的javascript工具库，压缩后不到5KB。

View on GitHub Download .zip Download .tar.gz

Install 安装

$ bower install fine.js

Compatibility 兼容性

Compatible with IE8+, Chrome, Firefox.
与IE8及以上浏览器、Chrome、Firefox兼容。

API 编程接口

Element 元素

$(selector, [context])

Description
描述		 Method of accessing DOM elements using CSS selectors, base on native document.querySelectorAll function. 使用CSS选择器访问DOM元素的方法，基于原生的document.querySelectorAll函数。
Parameter
参数		 selector: CSS selector. CSS选择器。
[context]: Optional, define selection range, default value is document. 可选参数，选择器访问范围，默认值document。
Usage
示例		 
//Get elements with class name 'btn'. 获取类名为 'btn' 的所有元素。
$('.btn');

//Get input elements in div whose id is 'div1'. 获取id为 'div1' 的元素内的所有input元素。
$('input', '#div1')

$.hasClass(el, className)

Description
描述		 Method of checking if element has a class. 检查元素是否包含一个类的方法。
Parameter
参数		 el: Single element or elements(HTMLCollection or NodeList type). 单个元素或元素集（HTMLCollection 或者 NodeList 类型）。
className: Name of the class. 类名。
Usage
示例		 
//Check the first button if has class 'btn'. 检查第一个button元素是否包含类 'btn'。
$.hasClass($('button')[0], 'btn');

//Check all button elements if at least one has class 'btn'. 检查所有的button元素是否至少有一个包含类 'btn'。
$.hasClass($('button'), 'btn');

$.addClass(el, className1, className2,...,classNameN)

Description
描述		 Method of adding class to element. 给元素添加类的方法。
Parameter
参数		 el: Single element or elements(HTMLCollection or NodeList type). 单个元素或元素集（HTMLCollection 或者 NodeList 类型）。
classNameN: Name of the class. 类名。
Usage
示例		 
//Add class 'btn' to the first button element. 给第一个button元素添加类 'btn'。
$.addClass($('button')[0], 'btn');

//Add class 'btn', 'btn-info' and 'btn-lg' to all buttons. 给所有button元素添加'btn', 'btn-info' 和 'btn-lg' 类。
$.addClass($('button'), 'btn', 'btn-info', 'btn-lg');

$.removeClass(el, className1, className2,...,classNameN)

Description
描述		 Method of removing class to element. 给元素移除类的方法。
Parameter
参数		 el: Single element or elements(HTMLCollection or NodeList type). 单个元素或元素集（HTMLCollection 或者 NodeList 类型）。
classNameN: Name of the class. 类名。
Usage
示例		 
//Remove class 'btn' from the first button element. 从第一个button元素上移除类 'btn'。
$.removeClass($('button')[0], 'btn');

//Remove class 'btn', 'btn-info' and 'btn-lg' from all buttons. 从所有button元素上移除'btn', 'btn-info' 和 'btn-lg' 类。
$.removeClass($('button'), 'btn', 'btn-info', 'btn-lg');

$.empty(el)

Description
描述		 Method of emptying element (remove all child nodes). 清空元素（移除元素所有子节点）的方法。
Parameter

String 字符串

$.stringFormat(str, args)

Description
描述		 Method of generating formatted string with template string and argument array. 根据模板字符串和参数数组生成格式化后的字符串的方法。
Parameter
参数		 str: The template string with placeholders, the placeholders like '{0}', '{1}'... and '{-1}' return '{', '{-2}' return '}'. 包含占位符的模板字符串，占位符形如'{0}', '{1}'... 并且 '{-1}' 固定返回 '{', '{-2}' 固定返回 '}'。
args: The argument array to replace the placeholders within template string. 用来替换模板字符串中占位符的参数数组。
Usage
示例		 
var temp = 'The {0}.js less than {1}KB with compression, you can install it with {2}. {-1}^_^{-2}';
var info = $.stringFormat(temp, ['fine', '5', 'bower']);
//Result: The fine.js less than 5KB with compression, you can install it with bower. {^_^}
alert(info);

$.parseHTML(str)

Description
描述		 Method of parsing HTML string into HTMLCollection. 转换HTML字符串到HTMLCollection的方法。
Parameter
参数		 str: HTML string. HTML字符串。
Usage
示例		 
//Get elements. 得到节点。
var els = $.parseHTML('<div>Hello fine.js!</div>')[0];
//Append element to body. 将节点添加到body中。
$('body')[0].appendChild(els);

$.url([url])

Description
描述		 Method of getting url info. 获取url信息的方法。
Parameter
参数		 [url]: Optional, the url address, if undefined will pass the current page url. 可选参数，url地址，如果为空则获取当前页面的url。
Usage
示例		 
var url = $.url('https://example.com/index.html?lang=zh-cn&tab=show#abc');
//Result: source url is: https://example.com/index.html?lang=zh-cn&tab=show#abc
alert('source url is: ' + url.source);
//Result: protocol is: https
alert('protocol is: ' + url.protocol);
//Result: host is: example.com
alert('host is: ' + url.host);
//Result: port is:
alert('port is: ' + url.port);
//Result: query is: ?lang=zh-cn&tab=show
alert('query is: ' + url.query);
//Result: parameter 'lang' is: zh-cn
alert('lang is: ' + url.params.lang);
//Result: file is: index.html
alert('file is: ' + url.file);
//Result: hash is: abc
alert('hash is: ' + url.hash);
//Result: path is: /index.html
alert('path is: ' + url.path);
//Result: relative is: /index.html?lang=zh-cn&tab=show#abc
alert('relative is: ' + url.relative);
//Result: first segments is: index.html
alert('first segments is: ' + url.segments);

Array&Object 数组和对象

$.each(obj, fn)

Description
描述		 Common method of iterating array or attributes in object. 遍历数组或对象属性的通用方法。
Parameter
参数		 obj: The object need to be iterated, support Array, object, HTMLCollection, NodeList. 需要遍历的对象，支持数组、对象、HTMLCollection、NodeList。
fn: The callback function on each iterate, 1st parameter of it is index or attribute name, 2nd is iterate element or attribute value, 'this' in function is the iterate element. 每次遍历的回调函数， 函数的第1个参数是序号或属性名称，第2个参数是本次遍历到的元素或属性值， 函数内部的'this'指向本次遍历到的元素或属性值。
Usage
示例		 
//literate Array 遍历数组
var names = ['Sky', 'Tom', 'Nancy'];
$.each(names, function(i, item){
    alert('Hello! ' + item);
});

//literate NodeList 遍历NodeList
var inputs = document.querySelectorAll('input');
$.each(inputs, function(){
    alert(this.value);
});

//literate Object 遍历对象
var obj = {
    name: 'Sky',
    gender: 'Male'
};
$.each(obj, function(i, item){
    alert(i + ' is ' + item);
});

Description
描述		 Method of extending the first parameter object with others objects from the second parameters. 使用从第2个参数开始的对象来扩展第一个参数位置对象的方法。
Parameter
参数		 objN: Object. 对象。
Usage
示例		 
//Clone object. 复制一个对象。
var obj1 = $.extend({}, {name: 'Sky', gender: 'Male'});

//Extend others to the first object. 扩展其他对象到第一个对象中。
var obj2 = $.extend({}, {name: 'Sky', gender: 'Male'}, {name: 'Tom'}, {age: 20});
//Result: Object { name="Tom",  gender="Male",  age=20}

Request 请求数据

$.ajax(object)

Description
描述		 Method of loading remote data with http request. 通过http请求加载远程数据的方法。
Parameter
参数		 The parameter is object, the following is the attributes in it. 参数是一个对象，包括如下属性。
url: The remote request url. 远程请求地址。
[type]: Optional, the type of request, 'Get' or 'Post', default is 'Get'. 可选参数，请求的类型，'Get' 或者 'Post', 默认值为 'Get'。
[async]: Optional, define if async request, true or false, default is true. 可选参数，定义是否异步请求，true或false，默认值为true。
[data]: Optional, the data to be sent, default is null. 可选参数，需要被发送的数据，默认值为null。
[success]: Optional, the callback method when request succeed. 可选参数，请求成功后的回调函数。
[error]: Optional, the callback method when request failed. 可选参数，请求失败后的回调函数。
[complete]: Optional, the callback method when request completed. 可选参数，请求完成后的回调函数。
Usage
示例		 
$.ajax({
    url:'js/fine.js',
    success: function(data){
        alert(data);
    }
});

$.jsonp(url)

Description
描述		 Method of loading remote data with jsonp type, support for cross domain. 使用jsonp方式加载远程数据的方法，支持跨域。
Parameter
参数		 url: The remote request url. 远程请求地址。
Usage
示例		 
//Sending request. 发送请求。
$.jsonp('http://www.example.com/?callback=myJsonpCallback');

//Custom callback function, should be the same as the name from server. 自定义的回调函数，必须跟服务器返回的函数名一致。
function myJsonpCallback(data){
    alert(data);
}
Event 事件

$.on(element, eventName, handler)

Description
描述		 Method of adding event to element. 给元素添加事件的方法。
Parameter
参数		 element: Single element or elements(HTMLCollection or NodeList type). 单个元素或元素集（HTMLCollection 或者 NodeList 类型）。
eventName: The event name. 事件名称。
handler: The handler function when trigger the event. 触发事件时的处理函数。
Usage
示例		 
//Handler function. 处理函数。
var clickHandler = function(event){
    //Other codes...
}

//Add click event to the button. 给按钮添加点击事件。
$.on($('#btn1')[0], 'click', clickHandler);

$.off(element, eventName, handler)

Description
描述		 Method of removing event from element. 从元素上移除事件的方法。
Parameter
参数		 element: Single element or elements(HTMLCollection or NodeList type). 单个元素或元素集（HTMLCollection 或者 NodeList 类型）。
eventName: The event name. 事件名称。
handler: The handler function when trigger the event. 触发事件时的处理函数。
Usage
示例		 
//Remove click event from the button. 从按钮上移除点击事件。

$.getEvent(event)

Description
描述		 Method of getting event object in handler function. 获取处理函数中event对象的方法。
Parameter
参数		 event: The parameter in handler function. 处理函数的参数。
Usage
示例		 
//Handler function. 处理函数。
var clickHandler = function(event){
    //Get real event object
    event = $.getEvent(event);
    //Result: click
    alert(event.type);
    //Other codes...
}

//Add click event to the button. 给按钮添加点击事件。
$.on($('#btn1')[0], 'click', clickHandler);

$.getTarget(event)

//Add click event to the button. 给按钮添加点击事件。
